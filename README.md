# Withkid
> power by qasar framework v0.17.18

## Installation

OS X & Linux:

```sh
npm instal
```

Windows:

```sh
npm instal
```

## start the development serve (using quasar-cli)

```sh
quasar dev
```

## Build for spa
```sh
quasar build
```

[more infomation](https://quasar-framework.org/guide/quasar-cli.html)
